# lecture-ai-engineering
AIエンジニアリング実践講座（公開用リポジトリ）

## 概要
このリポジトリはAIエンジニアリングの実践的なコースの公開リポジトリです。AIエンジニアリングに関連する実践的な内容とサンプルコードが含まれています。

## ディレクトリ構造
- `day1/`: 1日目のコース内容
  - 基本的なAIエンジニアリングの実践
  - モデルデプロイメントの例
  - コード最適化の実践

## 使用方法
1. リポジトリのクローン
```bash
git clone https://github.com/your-username/lecture-ai-engineering.git
```

2. 依存関係のインストール
```bash
pip install -r requirements.txt
```

3. サンプルの実行
```bash
python day1/example.py
```
